## Descripción

Este proyecto fue desarrollado en Python aplicando los conceptos fundamentales de Programación Orientada a Objetos (POO).

El sistema permite registrar productos disponibles en un restaurante y clientes registrados. Para su desarrollo se utilizaron clases, objetos, atributos, métodos, constructor (__init__), método especial (__str__) e importaciones entre módulos.

## Estructura del proyecto

restaurante_app/
│
├── modelos/
│   ├── producto.py
│   └── cliente.py
│
├── servicios/
│   └── restaurante.py
│
├── main.py
│
└── README.md

## Funcionalidades

- Registrar productos.
- Registrar clientes.
- Mostrar productos disponibles.
- Mostrar clientes registrados.
- Aplicar organización modular mediante carpetas y archivos independientes.

## Reflexión

La modularidad permite organizar mejor el código y separar responsabilidades entre los diferentes componentes del sistema. Esta práctica facilita el mantenimiento, la reutilización y la escalabilidad de los programas, además de mejorar la comprensión del proyecto por parte del desarrollador.
